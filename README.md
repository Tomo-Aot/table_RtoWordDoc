# Rで作成した表をワードファイルに出力

卒業研究などで表を作成する際、
エクセルを使用する人が多いと思います。
ここでは、Rで解析した結果を表にまとめる際に、
直接文書ファイルに出力する方法を共有しようと思います。

## 表の作成

はじめに、出力する表をRで作成します。
使用するパッケージを読み込みます。

```
library(tidyverse)
library(gt)
```

次に、表にまとめたい情報をデータフレームにまとめます。
データフレームにまとめる際は、$tidyverse$ パッケージの
tibble() 関数を使用します。
この関数を使うことで、列名の型が表示されるので便利です。

```
list = tibble(
  sp = c("A", "B", "C"),
  value = c(100, 25,  60)
)
```

他のファイルを読み込んでRで使用する際は、
読み込んだデータフレームにパイプ演算子で
as_tibble() 関数に渡します。

```
data |> as_tibble()
```
